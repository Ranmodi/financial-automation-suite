# Module Map

| Module | Purpose |
|---|---|
| `margin_alerts` | Detects margin-related events and generates alert outputs. |
| `desk_balance_alerts` | Monitors desk-client balances and negative balances. |
| `idle_cash_alerts` | Identifies accounts with relevant idle cash. |
| `advisor_maturities` | Consolidates product maturities by advisor. |
| `advisor_profitability` | Builds advisor-level profitability summaries. |
| `corporate_workflow` | Generates corporate workflow communication outputs. |
| `nps_automation` | Public-safe NPS response export template. |
| `outlook_html_tools` | Generic Outlook HTML e-mail utility. |
| `shared` | Common configuration, Excel, e-mail and logging helpers. |
